IoT APPLICATIONS

The network of physical objects, or "things," that are embedded with sensors, software, and other technologies for the purpose of connecting and exchanging data with other devices and systems over the internet is referred to as the Internet of Things (IoT):

  • Smart Home IoT integration
  • Smart Lighting System
  • Weather Monitoring System
  • Agricultural Plant Monitoring
  • Smart Home Appliances IoT integration
  • Home Assistants

ACTIVITY 1: IoT Application

  1. IoT Application: It is an interface connected with IoT devices over Internet to collect data and make smart real-time decisions
  2. Examples: IoT Applications are used to control and know the status of Home appliances, Air conditioning system, Health monitoring, Agricultural plant health monitoring and more
  3. Features: It includes user friendly interface, access to real-time data, seamless computing, communication between devices, data security.

ACTIVITY 2: Smart Building

  1. Smart Building: Smart decisions are those that are made in real-time using algorithms to analyse sensor data and to connect Internet of Things devices.
  2. Smart Building Scenarios: Smart air conditioning that adjusts to the surrounding temperature, automated lighting, doors that open when someone walks by, a smart electric metre, and a biometric security system.
  3. Features: Features of smart building includes integration IoT devices, centralised monitoring and control system, power consumption data, user friendly interface and data security.

ACTIVITY 3: Control Interface

  1. Users: people who live in smart buildings, work in smart offices, and own smart businesses.
  2. Parts needed: Smartphones, microcontrollers, fingerprint sensors, LDR sensors, DHT22 sensors, and other smart devices.
  3. Features: Machine learning algorithms, a user-friendly interface, remote access, and real-time data collection.
